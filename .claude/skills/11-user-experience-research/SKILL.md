# User Experience Research

## Journey Mapping
Map user experience across stages: Awareness → Consideration → Purchase → Usage → Renewal

For each stage: Actions, Touchpoints, Thoughts, Emotions, Pain Points, Opportunities

## Usability Testing
**Process:** Recruit → Script → Moderate → Observe → Analyze → Report
**Metrics:** Task success rate, time on task, error rate, satisfaction

## Service Design
**Service Blueprint:** Frontstage (customer-facing) + Backstage (internal processes)

## Information Architecture
**Tools:** Card sorting, tree testing, site maps
**Principles:** Findability, understandability, scalability
