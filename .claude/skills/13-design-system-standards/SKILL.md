# Design System & Standards

## Design System Structure
**Foundations:** Colors, typography, spacing, icons
**Components:** Buttons, inputs, cards, modals
**Patterns:** Navigation, forms, data display
**Documentation:** Usage guidelines, code examples

## Accessibility (WCAG 2.1)
**Level A (minimum):** Alt text, keyboard navigation, color contrast
**Level AA (target):** 4.5:1 text contrast, captions, resizable text
**Level AAA (enhanced):** 7:1 contrast, sign language

## Design Critique Framework
**What:** Describe objectively
**Why:** Explain reasoning
**How:** Suggest improvements
**Balance:** 2 positives for every critique

## Dev Handoff
**Specs:** Spacing, sizing, colors (hex), typography (weights, sizes)
**Assets:** SVGs, images (2x, 3x for retina)
**States:** All component variations
**Behavior:** Interactions, animations, edge cases
