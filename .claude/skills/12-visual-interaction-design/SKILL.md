# Visual & Interaction Design

## Avoiding "AI Slop" Aesthetic

**Typography:** Choose distinctive fonts, avoid Arial/Inter
**Colors:** Unique palettes, not generic blues/grays
**Layout:** Asymmetric, distinctive compositions
**Details:** Custom illustrations, thoughtful micro-interactions

## Visual Hierarchy
- Size, color, contrast, spacing
- F-pattern and Z-pattern scanning
- Progressive disclosure

## Interaction Design
**Micro-interactions:** Feedback, state changes, delighters
**Transitions:** Ease, duration, choreography
**States:** Default, hover, active, disabled, loading, error

## Responsive Design
**Breakpoints:** Mobile (<768px), Tablet (768-1024px), Desktop (>1024px)
**Approach:** Mobile-first, progressive enhancement
