# Collaboration & Integration

## Cross-Functional Workflows

**PM → Design:**
- Input: PRD, user research, success metrics
- Output: Design specs, prototypes, user flows

**Design → Dev:**
- Input: Final designs, component specs, assets
- Output: Implemented features, edge cases identified

**Content → Design:**
- Input: Copy strategy, messaging, tone guidelines
- Output: Content-informed design, in-context copy

## Handoff Best Practices

**Documentation:**
- Clear objectives and context
- Success criteria
- Timeline and dependencies
- Open questions

**Communication:**
- Regular syncs (daily standups, weekly reviews)
- Async updates (Slack, docs)
- Decision logging (ADRs)

## Alignment Techniques

**Kickoff:** Align on goals, roles, timeline
**Check-ins:** Weekly progress, blockers, decisions
**Retros:** What worked, what didn't, improvements

## Collaboration Tools

**Design:** Figma (collaborative design)
**PM:** Jira, Linear (project tracking)
**Docs:** Notion, Confluence (knowledge base)
**Communication:** Slack (async), Zoom (sync)
