# Design Inspiration Sources

Curated platforms to find modern website design inspiration. Used by the web-design-research skill.

## Primary Platforms

| Platform | URL | Best For |
|----------|-----|----------|
| Godly | [godly.website](https://godly.website/) | Curated modern sites, dark/premium aesthetic |
| Awwwards | [awwwards.com](https://www.awwwards.com/) | Award-winning sites (SOTY, SOTD), scored by jury |
| Land-book | [land-book.com](https://land-book.com/) | Landing page inspiration, SaaS-heavy |
| SiteInspire | [siteinspire.com](https://www.siteinspire.com/) | Editorial, cultural, minimalist sites |
| Httpster | [httpster.net](https://httpster.net/) | Clean, understated modern design |
| Supahero | [supahero.io](https://supahero.io/) | Hero section inspiration specifically |

## Static/Visual Inspiration

| Platform | URL | Best For |
|----------|-----|----------|
| Dribbble | [dribbble.com/search/hero-section](https://dribbble.com/search/hero-section) | Hero section mockups, UI concepts |
| Behance | [behance.net](https://www.behance.net/) | Full case studies, branding + web |

## Design Trend References

| Source | URL | Best For |
|--------|-----|----------|
| frontend.horse | [frontend.horse](https://frontend.horse/) | CSS/animation trend analysis (e.g., "The Linear Look") |
| CSS Design Awards | [cssdesignawards.com](https://www.cssdesignawards.com/) | Scored awards, WOTY rankings |
| Rimbo Designs Top 50 | YouTube | Annual curated video rankings |
| slashdev.io | [slashdev.io](https://slashdev.io/) | "Websites that teach you design" lists |

## How to Use

1. Browse platforms weekly for new additions
2. When a site stands out → add to `sites.csv`
3. Take screenshots → save to `screenshots/{domain}/`
4. Update `web-design-playbook.md` if new patterns emerge
